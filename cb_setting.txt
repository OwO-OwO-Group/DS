確認 C:\Program Files (x86) 沒有 CodeBlocks
安裝 codeblocks-17.12mingw-setup.exe (全部下一步)
到 settings -> Complier -> GNU GCC Compiler -> Complier Flags -> General
	設定 Have g++ follow the C++11 ISO C++ language standard [-std=c++11]
(若沒有這樣的設定，慶下載最新的IDE)
雙擊 Group10.cbp 
(若沒有設定預設開啟程式，選擇 C:\Program Files (x86)\CodeBlocks\codeblocks.exe)
點擊 Build -> Build and Run
